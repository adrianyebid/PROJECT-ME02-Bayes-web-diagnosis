# Bayes Web Diagnosis

Proyecto base para implementar un sistema bayesiano de diagnostico probabilistico de fallas en una aplicacion web distribuida.

## Objetivo

Implementar desde cero una red bayesiana y un algoritmo de inferencia por enumeracion para calcular probabilidades posteriores de la forma P(X | evidencia).

## Ejecucion

python main.py

## Estructura inicial

* main.py
* node.py
* bayesian_network.py
* inference.py
* model.py
* scenarios.py
* utils.py
* documentation/

## Integrantes y modulos

* Integrante 1: planteamiento del problema y coordinacion.
* Integrante 2: marco teorico.
* Integrante 3: diseno de red bayesiana y CPT.
* Integrante 4: estructuras base.
* Integrante 5: inferencia.
* Integrante 6: escenarios de prueba.
* Integrante 7: manuales, menu y validacion final.
